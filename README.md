# 🎬 Movie Rating Normalizer

A web app that searches movies and normalizes ratings from multiple sources into one unified score.

## ✨ Features
- Search any movie by title
- Search movies by actor name
- Filter results by genre
- Sort by score, popularity or newest
- Normalized unified score out of 10
- Badge system — Great, Good, Average, Poor
- Click any movie for full details
- Budget, Revenue, Runtime, Genres
- Beautiful dark theme with animations

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- TMDB API

## 📊 How Scoring Works
| Source | Weight |
|--------|--------|
| TMDB Score | 60% |
| Popularity | 20% |
| Vote Confidence | 20% |

## 🚀 How To Use
1. Open `index.html` in your browser
2. Search any movie title or actor name
3. Filter by genre and sort by score
4. Click any card for full details
